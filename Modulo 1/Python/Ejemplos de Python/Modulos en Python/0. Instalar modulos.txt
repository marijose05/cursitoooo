Comandos de Instalación para Cada Módulo

Para instalar NumPy:
pip install numpy

Para instalar Pandas:
pip install pandas

Para instalar Matplotlib:
pip install matplotlib

Para instalar Scikit-learn:
pip install scikit-learn

Para instalar Requests:
pip install requests

Puedes instalar varios paquetes a la vez separándolos con un espacio:
pip install numpy pandas matplotlib scikit-learn requests
